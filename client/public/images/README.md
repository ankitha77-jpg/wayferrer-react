# Image Management Guide

This folder contains all the images used on your Wayfarer Footprints website. You can easily update any image by replacing the file with a new image that has the same filename.

## How to Replace an Image

1. **Find the image you want to replace** in this `/images` folder
2. **Prepare your new image**:
   - Save it with the exact same filename as the existing image
   - Recommended image formats: JPG, PNG, WebP
   - Keep file sizes under 2MB for optimal website performance
3. **Replace the file**: Upload your new image to this folder, overwriting the existing file
4. **Refresh your website** to see the changes

## Image Naming Convention

All images are organized with descriptive names:
- `logo.png` - Website logo in navigation
- `hero-background.jpg` - Main homepage background image
- `profile-photo.jpg` - Your profile photo
- `book-cover.jpg` - Book cover image
- `destination-[name].jpg` - Destination guide images
- `testimonial-[number].jpg` - Customer testimonial photos

## Image Size Recommendations

- **Logo**: 200x200 pixels (square)
- **Hero backgrounds**: 1920x1080 pixels (landscape)
- **Profile photos**: 400x400 pixels (square)
- **Destination images**: 800x600 pixels (landscape)
- **Blog post images**: 1200x800 pixels (landscape)

## Important Notes

- Always keep the same filename when replacing images
- Images will automatically appear on your website after replacement
- For best performance, optimize images before uploading
- If you need help, contact your web developer

## Backup Reminder

Keep copies of your original images in a separate backup folder before making changes.
